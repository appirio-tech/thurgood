A Warm Welcome!

Thurgood Manager is the best way to manage the Thurgood API. Built from fluffy bits of 0's and 1's, with support from AngularJS and Bootstrap, it just screams slickness and performance. Because an awesome API deserves an awesome app!

Instructions

  • Login with your Appirio credentials.
  • Create a new job or search for an existing job by keyword from the "Jobs" menu.
  • Once you create your job and ensure that it looks correct, submit the job for processing.
  • While the job is processing, click the "Event Viewer" button to open the log viewer to see the process of the job.
  • When creating a job, if you select "email" as the notification method, you will receive an email once the job has completed.
  • You can submit a job as many times as you would like. Simpy upload new code to the same URL or change the URL of the job.
  • When creating a job you can specify the "steps" to perform. Supported steps are "all" and "scan". The "scan" step will just peform a Checkmarx scan while "all" will perform a Checkmarx scan plus build, test and deploy. Default is "all".
  • You can also specify a project for your job and tie the build process to a specific server (e.g., salesforce org). Contact jeff@appirio.com to set this up.

Take a look at the source code or the API documentation if you need more info.
