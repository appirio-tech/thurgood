A Warm Welcome!

Thurgood Manager is the best way to manage the Thurgood API. Built from fluffy bits of 0's and 1's, with support from AngularJS and Bootstrap, it just screams slickness and performance. Because an awesome API deserves an awesome app!

Why it rocks

  • Beautiful design using Flatly from Bootswatch.com
  • Table can be sorted, includes search, pagination and choice of displayed items
  • Job's detail page included; when you press back the previous table state is restored
  • Module-based structured app, endpoint is encapsulated as a factory resource (in services.js)
  • Loading and errors both handled within an alert
